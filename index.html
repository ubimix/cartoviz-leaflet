Fonctions et API

  • Alternative libre à Google Maps JS
  • Simplicité à l'intérieur / à l'extérieur
  • Noyau + extensions
  • Multi-devices / standalone

Interactivité

  • Fond de carte (tuiles avec design personnalisé)
  • Interactivité (zoom, pan, click)
  • Figures géométriques (points, lignes, polygones)

Ecosystème

  • Mapbox, CartoDB
  • Foursquare, Evernote
  • Nombreux plugins

Exemple : afficher un fond de carte

<script type='text/javascript'>
var map = L.map('map').setView([ 48.86167, 2.33314], 13);
L.tileLayer('http://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    attribution: 'Map data © <a href="http://openstreetmap.org">OpenStreetMap</a>',
    maxZoom: 18
}).addTo(map);
</script>

Exemple : ajouter un marqueur

<script type='text/javascript'>
 var marker = L.marker([ 48.86167, 2.33314]).addTo(map);
</script>

Exemple : clusteriser des données GeoJSON

<script type='text/javascript'>
var DATA = {"type": "FeatureCollection","features":[
    {"type": "Feature",
     "geometry": {"type": "Point","coordinates": ["2.3475","48.853611111111114"]},
     "properties": {"name": "Crypte archéologique du parvis Notre-Dame"
  }},...]};
var markers = new L.MarkerClusterGroup();
for (var i = 0; i < DATA.features.length; i++) {
    var coords = DATA.features[i].geometry.coordinates;
    var latlng = L.latlng(coords[1],coords[0]);
    var marker = new L.Marker(latlng);
    markers.addLayer(marker);
}
map.addLayer(markers);
</script>

Combinaison de Leaflet et de D3

  • Leaflet pour gérer le fond de carte et les données géolocalisées
  • D3 pour animer et expliquer les données : statistiques, dataviz

3 modes d'utilisation de D3 avec Leaflet :

  • Comme librairie de pré-traitement des données
  • En couche séparée pour animer les éléments individuels sur Leaflet
  • D3 pour visualiser les données
